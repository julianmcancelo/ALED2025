Crear Primer Usuario Administrador

Parece que es la primera vez que se inicia la aplicación. Por favor, crea el usuario administrador inicial.

Nombre Completo El nombre es obligatorio. Correo Electrónico El correo es obligatorio. El formato del correo no es válido. Contraseña La contraseña es obligatoria. La contraseña debe tener al menos 6 caracteres.
